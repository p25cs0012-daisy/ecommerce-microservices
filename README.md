# ecommerce-microservices
A microservices e-commerce platform


Topic:
E-Commerce Microservices Platform with Real-Time Payment Integration
Abstract:
This project develops a distributed e-commerce platform using microservices architecture where five independent services (User, Product, Order, Payment, Notification) communicate via REST APIs. The system uses techstack that will demonstrate scalability through independent service scaling, reliability through data persistence and secure payments, and high availability via distributed design eliminating single points of failure.
Technology Stack:
Frontend: React.js, Backend: Node.js + Express.js, Database: MongoDB
Authentication: JWT, Payments: Razorpay
Quality Attributes:
Scalability, Reliability, Security, Availability
